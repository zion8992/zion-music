## Zion's Music
A Mindustry mod that revamps the default Mindustry soundtrack.<br>
**NOTE:** This mod is a work in progress, Im still adding new songs and improving existing ones.<br>
**Changelog**: [changelog.txt](changelog.txt)<br>
<br>
## Musician's Notes
All songs were made with FL Studio using synths and custom instruments.<br>
**game1.ogg**<br>
Inspired by game4.ogg from the original soundtrack.<br>
<br>
**game2.ogg**<br>
The first soundtrack you hear when joining a game. I decided to use a Low Pass EQ and Phaser in certain parts of the song.<br>
<br>
**game4.ogg**<br>
Inpsired by game1.ogg from the original soundtrack.<br>
<br>
**game3.ogg**<br>
Not inspired by anything.<br>
<br>
**menu.ogg** (main menu)<br>
Less dissonant and less "mindustry" than the other songs.<br>
<br>
**editor.ogg**<br>
Inspired by 'editor.ogg' (original soundtrack) and 'Impact' (Purrple Cat: Distant Worlds 4)<br>
